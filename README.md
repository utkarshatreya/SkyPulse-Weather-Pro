# 🌤️ SkyPulse Weather Pro

SkyPulse Weather Pro is a modern and responsive weather application built using HTML, CSS, and JavaScript. It provides real-time weather information, hourly forecast, and a clean user interface with a beautiful background.

## 🚀 Live Demo

🔗 https://utkarshatreya.github.io/SkyPulse-Weather-Pro/

---

## 📌 Features

- 🌍 Search weather by city name
- 🌡️ Real-time temperature updates
- ☁️ Weather condition with icons
- 💧 Humidity information
- 💨 Wind speed details
- 🌅 Sunrise and Sunset timings
- 👀 Visibility information
- 🌡️ Feels Like temperature
- 📊 Air Pressure details
- 📅 Date and Live Digital Clock
- 📱 Fully Responsive Design
- 🎨 Glassmorphism UI with attractive background
- ⚡ Fast and lightweight

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API
- GitHub Pages

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/utkarshatreya/SkyPulse-Weather-Pro.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
